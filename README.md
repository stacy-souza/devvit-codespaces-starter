# Devvit Codespaces Starter

This is a beginner-friendly **Devvit app starter** that runs entirely in your browser.

No local installs. No setup. Works on any computer.

---

## First time only

1. Click **Use this template** to create your own repository
2. Give it a name and make sure **you** are the owner
3. Click **Create repository**

---

## Open in Codespaces

1. Go to your **new repository**
2. Click **Code â†’ Codespaces tab**
3. Click **Create new Codespace**
4. Wait about 1 minute for setup

---

## First run

In the terminal at the bottom, type:

```bash
npm start
```
If prompted, log in to Devvit:

```bash
devvit login
```


You only need to do this once.
